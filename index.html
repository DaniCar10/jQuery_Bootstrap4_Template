Fazemos Frontend

Este template foi construído para seguir os estudos em frontend, aperfeiçoando os conhecimentos nos frameworks Bootstrap 4 e jQuery. Foram utilizados diversos plugins. Destaque para a animação Parallax, o Contador e o Formulário dinâmico, com uso de máscara e consulta de CEP. Também podemos destacar no trabalho o código jQuery para a área de Perguntas Frequentes e o CSS animado para incrementar a seção Equipe. Plugin ScrollReveal anima os elementos para o usuário.
